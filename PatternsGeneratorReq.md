图像大小有两种：

​	大尺寸：2000*2000， 在密集图案时使用

图像格式：

​	.png, 像素值从0~255之间，大部分像素为0或255，边缘至多有1个像素宽的过渡区（抗锯齿）

代码语言：

​	python

## 2. 三类数据说明

### 2.1 Line Space

#### 图案说明

Line Space图案，在没有旋转角度的情况下，为横线和空白形成的周期图案。横线和空白占的像素数相等。如第0-9行为1，10-19行为0，就是一个HP为10像素的间距比为1：1的LineSpace图案。

#### 关键特征

| Key                         | Value             |
| --------------------------- | ----------------- |
| 旋转角度（横线旋转角度为0） | [0:5:180]°        |
| HP大小                      | 5个像素到60个像素 |
| 间距比（1区域：0区域）      | 1（保留可扩展性） |
| 区分孤立和密集              | True              |

### 2.2 Contact Hole

#### 图案说明

Contact Hole图案，有两种

#### 方形hole 

在没有旋转角度的情况下，为一个边长为HP的正方形，其内区域都是1。间距比为：hole的边长和hole之间的间距比，默认为1:1（即hole尺寸=间距）

##### 关键特征

| Key                    | Value             |
| ---------------------- | ----------------- |
| 旋转角度               | [0:0.05:1]°       |
| HP大小                 | 5个像素到60个像素 |
| 间距比（1区域：0区域） | 1（保留可扩展性） |
| 区分孤立和密集         | True              |

#### 圆形hole

为一个圆形，直径为HP的圆。间距比为：hole的直径和hole之间的间距（两圆之间的最短距离） 比，默认为1:1（即直径=间距）

#### 关键特征

| Key                    | Value             |
| ---------------------- | ----------------- |
| HP大小                 | 5个像素到60个像素 |
| 间距比（1区域：0区域） | 1（保留可扩展性） |
| 区分孤立和密集         | True              |

### 2.3 others

其他的类型，除了2.1和2.2的内容；包含以下3种图案：

- **Z字形（Zigzag）**：锯齿状折线周期图案
- **米字型（Star/Cross）**：横、竖、两条对角线交叉形成的米字形周期图案
- **菱形网格（Diamond grid）**：菱形周期排列图案

#### 关键特征

| Key                              | Value             |
| -------------------------------- | ----------------- |
| 旋转角度                         | [0:5:180]°        |
| HP大小                           | 5个像素到60个像素 |
| 间距比（你需要根据图案内容定义） | 1（保留可扩展性） |
| 区分孤立和密集                   | True              |
